# dartDevTools
this project would contain some dev tools for the language dart, such as logger, you are more then welcome to contribute and add your developing tools for dart (by developing tools I mean sub project's that can help other developers with common problem's, like validate eMail by regax or users system)

## logger
the sub-project "logger" is based on the "logger" project from github; git "https://github.com/leisim/dartx.git", the original "logger" project was created by "Simon Leier, leisim"
